# Angular Maintenance

This workflow ensures the Angular project is clean, formatted, and follows best practices.

## Steps

1. **Check Linting**
   - Run `npm run lint`

2. **Format Code**
   - Run `npm run format`

3. **Verify Build**
   - Run `npm run build` (optional, for validation)

4. **Verify Application**
   - Open browser and check critical paths (Login, Checkout, Profile).
